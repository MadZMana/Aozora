# Aozora Discord Bot

[![Node.JS](https://img.shields.io/badge/Made%20with-Node.js-blue.svg)](https://nodejs.org/en)[![Current Version](https://img.shields.io/badge/version-22.20.0-green.svg)](https://nodejs.org/en/download)

Aozora is a Discord bot with a collection of random features, updated over time.

---

## Features

- Instagram Video Sender

- TikTok Video Sender

---

## Setup
- Clone this repo to your desktop and run `npm install` to install all the dependencies.

- Make your Discord Bot [here](https://discord.com/developers/applications)

---

## Usage

- Rename `.env.example` to `.env` and edit it with your credentials.

- Add the bot to your Discord server.

- You can run `npm start` to start the bot.

- Once your bot started, type `/igdl <instagram_url>` to send Instagram reels in your Discord server and type `ttdl <tiktok_url>` to send TikTok videos.
